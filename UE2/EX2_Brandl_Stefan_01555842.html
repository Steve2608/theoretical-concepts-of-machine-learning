Answer 1.1:

Do you know the name of the distribution to which the pdf belongs?

It is a special case of the general binomial distribution - the Bernoulli Distribution: $$ \binom{n}{k} p^k (1 - p)^{n - k} ~~~~ \text{with $n=1$ and $k \in \{0, 1\}$} \\ 1 p^k (1 - p)^{1 - k} ~~~~ \text{renaming $k$ to $x$} \\ p^x (1 - p)^{1 - x} $$

Calculate the Fisher information! \begin{align*} I_F(p) &= -\mathbb{E}_X \big[\frac{\partial ^2}{\partial p^2} \log f(x;p) \big] \end{align*}

Firstly, we calculate the first derivative: $$ \frac{\partial}{\partial p}[x \ln(p) + (1-x)\ln(1-p)] = \frac{p - x}{(p - 1) p} $$

the second derivative: $$ \frac{\partial}{\partial p}[\frac{p - x}{(p - 1) p}] = -\frac{p^2 - 2xp + x}{(p - 1)^2 p^2} $$

Negative expectation: \begin{align*} I_F(p) &= -\mathbb{E}_X \big[-\frac{p^2 - 2xp + x}{(p - 1)^2 p^2}\big] \\ &= \mathbb{E}_X \big[ \frac{p^2 - 2xp + x}{(p - 1)^2 p^2} \big] \\ &= \frac{-p^2 + p}{(p - 1)^2 p^2} = \frac{-p (p - 1)}{(p - 1)^2 p^2}\\ &= -\frac{1}{p(p - 1)} = \frac{1}{p(1 - p)} \end{align*}

import numpy as np
from numpy.random import normal
import matplotlib.pyplot as plt

def draw(n, loc=5, scale=2):
    return normal(loc, scale, n)

def est1(data):
    return np.sum((data - data.mean())**2) / (len(data) - 1) 

def est2(data): 
    return np.sum((data - data.mean())**2) / len(data) 

def plot_stats(d1, d2, l1, l2, e, l3, title):
    plt.figure(figsize=(16, 7))
    plt.plot(d1, linewidth=1)
    plt.plot(d2, linewidth=1)
    plt.hlines(e, xmin=0, xmax=len(d1), linewidth=1, linestyle='--')
    plt.legend([l1, l2, l3])
    plt.title(title)
    plt.show()

Answer 2.2:

On average, which estimator is closer to the true value? And which one has less variance?

The first estimator $\hat\sigma_1$ is closer to the real value (given by the CRLB). Since the variance is computed identically (only the factors $\frac{1}{n-1}$ and $\frac{1}{n}$ are different) estimator $\hat\sigma_2$ has a lower variance (bigger divisor).

What is the Cramer-Rao Lower Bound for this estimation task? You will probably have observed that one of the estimators has even less variance than the CRLB predicts. How can that be?

The CRLB is the inverse of the fisher information matrix:

mean: $\frac{1}{\frac{n}{\sigma^2}} = \frac{\sigma^2}{n} = \frac{4}{n}$

variance: $\frac{1}{\frac{n}{2 \sigma^4}} = \frac{2 \sigma^4}{n} = \frac{32}{n}$

The second estimator $\hat\sigma_2$ predicts a lower value than the CRLB predicts, because it is biased.

The variance can be defined as follows:

\begin{align*} \sum_i^n (x_i - \overline{x})^2 &= \sum_i^n x_i^2 - 2x_i\overline{x} - \overline{x}^2 \\ &= \sum_i^n x_i^2 - n\overline{x}^2 \end{align*}\begin{align*} \mathbb{E}(\sigma^2) &= \mathbb{E}(\sum_i^n \frac{(x_i - \overline{x})^2}{n - 1}) \\ &= \frac{1}{n - 1} \mathbb{E}(\sum_i^n x_i^2 - n\overline{x}^2) \\ &= \frac{1}{n - 1} \sum_i^n (\text{Var}(x_i) - n\text{Var}(\overline{x})) \end{align*}

We also know: \begin{align*} \text{Var}(x_i) = \sigma^2 & & \text{Var}(\overline{x}) = \frac{\sigma^2}{n} \end{align*}

Thus: \begin{align*} \mathbb{E}(\sigma^2) &= \frac{1}{n - 1} \sum_i^n (\text{Var}(x_i) - n\text{Var}(\overline{x})) \\ &= \frac{1}{n - 1} \sum_i^n (\sigma^2) - n\frac{\sigma^2}{n} \\ &= \frac{1}{n - 1} (n\sigma^2 - \sigma^2) = \frac{1}{n - 1} (n-1)\sigma^2 = \sigma^2 \end{align*} So we can see that the expectation of the variance equals the true variance.

For $\frac{1}{n}$ however: \begin{align*} \mathbb{E}(\sigma^2) &= \frac{1}{n} (n-1)\sigma^2 = \frac{n-1}{n} \sigma^2 \end{align*} We can see that this, however, is not the real variance - thus the estimator $\hat\sigma_2$ is biased.
